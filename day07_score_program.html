1. 今天目标

今天的重点不是再认识新命令，而是把前面学过的内容真正串起来。你要感受到：程序是在接收信息、处理信息、再给出结果。

2. 完整程序

score = int(input("请输入分数：")) if score >= 60: print("及格") else: print("不及格")
为什么这个程序重要： 它不是单个知识点，而是把第一周最关键的几块基础真正接起来了。

3. 逐行拆解

1

`input()` 获取输入

程序先问用户输入分数。这说明程序已经开始和人互动了。

2

`int()` 做类型转换

因为输入默认是文字，所以要把它转换成整数，后面才能做比较。

3

`score` 保存数据

把转换后的分数保存到变量里，后面判断时就能直接使用。

4

`if / else` 做判断

如果分数大于等于 60，就输出“及格”；否则输出“不及格”。

4. 程序完整流程

把这个程序理解成一条流程线，会更容易看到它为什么已经像一个完整工具。

用户输入分数 转换成整数 保存到变量 判断是否及格 输出结果
这就是程序闭环： 接收信息 → 处理信息 → 输出结果。哪怕只是一个简单的成绩判断，它也已经具备程序最核心的结构了。

5. 可以怎么升级

Up

从“是否及格”升级到“等级判断”

你后面可以继续扩展：90 分以上显示优秀，80 分以上显示良好，60 分以上显示及格，否则显示不及格。

7. 今日总结

  • `input()`：获取输入。
  • `int()`：把文字转换成整数。
  • 变量：保存分数。
  • `if / else`：判断条件。
  • `print()`：输出结果。
今天过关标准： 你能自己写出一个“输入数字 → 判断条件 → 输出结果”的小程序，就说明第一周最关键的基础已经搭起来了。

第一周结束后，你已经不是只会看代码的人了。

你现在已经能保存信息、区分类型、接收输入、做计算、做判断，还能把它们组合成一个完整程序。接下来最值得做的是继续扩展练习，或者补更多独立学习页。
