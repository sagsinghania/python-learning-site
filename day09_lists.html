1. 今天目标

你今天要理解的核心是：当数据不止一个时，不要再急着创建一堆零散变量。列表会帮你把同类数据放进同一个结构里，后面处理会方便很多。

2. 列表是什么

列表可以理解成一个有顺序的“数据盒子”。盒子里可以放多个值，而且这些值会按照先后位置排好。

例

最简单的列表

fruits = ["苹果", "香蕉", "橙子"]

这表示 `fruits` 里保存了 3 个水果名称。

想

它适合保存什么

一组名字、一周气温、多个分数、任务清单、购物车商品，都适合先用列表表示。

先记住这个外形： 列表通常写成 `[值1, 值2, 值3]`，外层是方括号。

3. 如何取值和修改

列表里的每个位置都有编号，这个编号叫索引。Python 从 0 开始数，所以第一个元素是 `0`，第二个是 `1`。

scores = [88, 92, 75] print(scores[0]) scores[2] = 80 print(scores)
  • `scores[0]` 取第一个分数。
  • `scores[2] = 80` 把第三个分数改成 80。
  • 索引写错位置，程序会报错。

4. 如何追加新内容

很多时候列表不是一开始就完整的，而是边用边加。最常见的方法是 `append()`。

tasks = ["背 5 个单词", "写 3 行 print"] tasks.append("完成今天打卡") print(tasks)
这一步很重要： 你会开始感觉到，程序里的数据不是固定不变的，它可以在运行过程中继续增长。

5. 列表为什么常和循环一起出现

列表里有多个值，而循环刚好擅长重复处理。它们放在一起时，程序就能把“名单里的每个人”“分数里的每一项”全部过一遍。

names = ["小林", "小周", "小王"] for name in names: print("欢迎你，", name)
拿到第 1 个名字 输出一次 拿到第 2 个名字 继续处理 直到列表结束

7. 今日总结

  • 列表是保存多个值的常见结构。
  • 列表写法是方括号包住多个元素。
  • 索引从 0 开始。
  • 可以取值、修改，也可以用 `append()` 追加。
  • 列表和循环配合后，能批量处理一组数据。
今天过关标准： 你能创建一个列表、取出其中一项、修改其中一项，并用循环把它们遍历一遍，就算今天掌握到位。
