1. 今天目标

今天你要理解的核心是：函数不应该只能写死一种结果。通过参数，我们可以把数据交给函数，让它根据不同输入执行相同结构但不同内容的逻辑。

2. 参数是什么

参数可以理解成“函数预留出来的输入位置”。调用函数时，把实际数据填进去，函数就知道该处理谁、该处理什么值。

def say_hello(name): print("你好，", name)
  • `name` 是参数名。
  • 它像一个临时变量，专门接收调用时传进来的数据。
  • 函数内部就能用这个参数继续做处理。

3. 定义时和调用时分别写什么

Def

定义函数时

写的是参数名。它表示“将来这里会收到一个值”。

def say_hello(name):
Call

调用函数时

写的是实际值。它表示“这次就把这个内容传进去”。

say_hello("小林")
定义参数位置 调用时传入值 函数内部使用这个值

4. 分数判断函数案例

把前面学过的条件判断和函数参数结合起来，就能写出一个更像工具的小功能。

def check_score(score): if score >= 60: print("及格") else: print("不及格") check_score(88) check_score(45)
这个例子很关键： `check_score` 的逻辑只写一次，但可以处理很多不同分数。这就是参数真正的价值。

5. 参数为什么重要

1

减少重复函数

不需要为小林、小周、小王分别写 3 个函数，一个带参数的函数就够了。

2

程序更灵活

同样的结构可以处理不同数据，这才是程序真正擅长的地方。

你可以把函数想成机器： 参数就是投进去的原料。原料不同，机器输出就不同，但机器本身的工作流程不需要重写。

7. 今日总结

  • 参数是函数预留出来的输入位置。
  • 定义函数时写参数名，调用时传实际值。
  • 参数让同一个函数可以处理不同数据。
  • 函数和参数结合后，程序会明显更灵活。
  • 函数参数是后面模块化写代码的重要基础。
