1. 今天目标

今天不是死背一堆方法名，而是先建立一个认识：字符串虽然看起来只是一段文字，但程序可以像处理列表一样处理它的长度、位置和部分内容。

2. 长度和单个字符

字符串里的每个字符也有位置。和列表类似，第一个位置依然从 0 开始。

word = "python" print(len(word)) print(word[0]) print(word[5])
  • `len(word)` 用来求长度。
  • `word[0]` 取第一个字符。
  • 下标超出范围会报错，所以要注意长度。

3. 切片是什么

切片就是从字符串里拿出一段连续内容。它的写法是 `[开始:结束]`，结束位置不包含在内。

text = "hello python" print(text[0:5]) print(text[6:12])
确定开始位置 确定结束位置 拿出中间这一段
先不要急着全背规则： 你只要先记住“开始包含，结束不包含”，练几次就会越来越顺。

4. 常见字符串方法

UP

`upper()` 和 `lower()`

把文字统一改成大写或小写，适合做输入标准化。

name = "PyThOn" print(name.lower())
TR

`strip()` 和 `replace()`

去掉首尾空格，或者把一段文字替换成别的内容。

text = " hello " print(text.strip())

5. 这些操作有什么用

用户输入的内容经常不规整，比如多了空格、大小写乱、需要截取前几个字符。这时字符串处理就派上用场了。

场景 1

整理用户名

输入前后多出的空格可以先用 `strip()` 去掉，再统一成小写。

场景 2

提取部分信息

例如从一串编号里取前 3 位分类码，或者从日期里切出年份部分。

你要意识到： 后面做交互程序时，处理输入文字和整理输出内容，几乎都会用到今天这些基础动作。

6. 今日练习

P1

P2

P3

整理输入格式

写一个字符串，前后故意加空格，再用 `strip()` 去掉空格，并用 `upper()` 或 `lower()` 统一格式。

7. 今日总结

  • 字符串也可以像数据结构一样按位置处理。
  • `len()` 用来求长度。
  • 切片能取出字符串中的一段。
  • `upper()`、`lower()`、`strip()`、`replace()` 都很常用。
  • 字符串处理最常用于整理输入和输出文本。
