核心命令

先掌握这些高频命令，你看很多入门代码时就不会再是完全陌生的状态。这里不要求你全背下来，只要求先看熟、先知道作用。

print() 输出

把内容显示到屏幕上

它的作用就是告诉程序：“把这句话、这个数字或者这个结果打印出来给我看。”

print("你好，Python")
input() 输入

从用户那里拿到信息

它会先停下来，等你输入内容，再把输入的东西交给程序继续处理。

name = input("请输入名字：")
type() 类型

查看数据属于什么类型

当你不确定一个值是字符串还是数字时，可以用它来检查。

print(type(18))
if 如果

满足条件时执行某件事

它像在问程序：“如果这个条件成立，那你就这样做。”

if score >= 60: print("及格")
else 否则

条件不成立时怎么办

当 `if` 里的条件不满足，就执行 `else` 里面的内容。

else: print("不及格")
for 循环

重复做同一类事情

当你不想把同样的代码写很多次时，就会用到循环。

for i in range(3): print("继续练习")

常用英文

这些英文会在入门阶段高频出现。你不用一次性全部背会，先做到“看见它就知道大概是什么意思”。

W1

数据相关

variable = 变量

可以把它理解成一个有名字的盒子。

value = 值

盒子里装着的内容就是值。

string = 字符串

被引号包起来的一段文字。

number = 数字

包括整数和小数。

W2

判断相关

condition = 条件

程序做判断时要先看条件。

True = 真

表示条件成立。

False = 假

表示条件不成立。

compare = 比较

比如大于、小于、是否相等。

W3

动作相关

output = 输出

让程序把结果显示出来。

input = 输入

从用户那里接收内容。

loop = 循环

重复做某件事。

function = 函数

把一段逻辑打包，方便重复使用。

Tip

零基础怎么记这些词更有效

不要把它们当成纯英语单词去硬背，而是把它们和“程序动作”绑在一起记。

  • `print` 一看到就想到“输出”。
  • `input` 一看到就想到“等用户输入”。
  • `if` 一看到就想到“如果满足条件”。
  • `for` 一看到就想到“重复做几次”。

把命令放进例子里理解

只看单词容易忘，把它们放进最短的程序里，你会更容易形成“这个命令是干什么的”直觉。

E1

输出一句欢迎语

先从最简单的开始，让程序说一句话。

print("欢迎开始学习 Python")
E2

问用户名字再打招呼

程序先拿到你的输入，再做回应。

name = input("请输入你的名字：") print("你好，" + name)
E3

判断是否及格

看见 `if` 和 `else`，就知道程序在做分支判断。

score = 75 if score >= 60: print("及格") else: print("不及格")
E4

重复输出三次

看见 `for`，就知道程序在重复处理。

for i in range(3): print("继续练习")
