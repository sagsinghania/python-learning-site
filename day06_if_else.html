Day 06 / If Else

程序不只是按顺序往下走，
它也开始会根据条件做不同选择。

今天学的 `if / else` 是真正把程序从“固定执行”带向“判断逻辑”的关键。它会让你的程序第一次像是在根据情况作决定。

先看是否成年案例 回看运算符
2 个分支 满足条件 / 不满足条件
1 个结构 `if ... else ...`
1 个重点 缩进必须写对

你今天会第一次理解

为什么说程序不是只会死板执行，它也能根据条件给出不同结果。

if age >= 18: print("已经成年") else: print("还未成年")

1. 今天目标

你今天要学会的，不只是把 `if` 和 `else` 打出来，而是真正理解：程序可以根据不同条件，走向不同分支。

2. 基本结构

if 条件: 条件成立时执行的代码 else: 条件不成立时执行的代码
一句话理解： 如果条件成立，就做 A；否则，就做 B。

3. 是否成年案例

最常见的例子之一，就是根据年龄判断是否成年。

age = 20 if age >= 18: print("你已经成年") else: print("你还未成年")
  • 如果 `age >= 18` 成立，就输出“已经成年”。
  • 如果不成立，就输出“还未成年”。

4. 条件判断在想什么

你可以把它理解成程序先问自己一个问题：这个条件是真是假？如果是真，走一边；如果是假，走另一边。

先比较 得到 True / False 进入不同分支 输出不同结果
这是程序思维的重要一步： 从今天开始，你写的代码已经不是只有一个固定答案，而是会根据情况变化了。

5. 为什么缩进重要

Python 里，缩进不是装饰，而是结构本身。缩进错了，程序就会报错，或者执行逻辑不对。

今天特别要注意： `if` 下面和 `else` 下面的代码要缩进，而且通常缩进层级要保持一致。

7. 今日总结

  • `if`：如果条件成立。
  • `else`：否则。
  • 条件判断让程序有了不同执行路径。
  • 缩进是 Python 结构的一部分，不能乱。
今天过关标准： 你能自己写出一个“根据条件输出不同结果”的程序，就算今天掌握了最核心的内容。

条件判断学会后，就能把前面几天真正串成一个小程序。
