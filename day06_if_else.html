1. 今日目标

条件判断的关键不是语法本身，而是“不同情况走不同分支”。

  • if 表示“如果满足条件”。
  • else 表示“如果不满足，就走另一条路”。

2. 核心理解

这一页先围绕“if / else”建立直觉。不要急着追求复杂，先把最小理解做稳。

C1

先理解用途

if 表示“如果满足条件”。

C2

再理解边界

else 表示“如果不满足，就走另一条路”。

3. 示例代码

下面这段代码是今天最值得看懂的最小例子。

score = 80 if score >= 60: print("及格") else: print("不及格")
这段代码在说明什么： 当条件成立时执行 if 里面的代码，不成立时执行 else 里面的代码。

4. 常见提醒

  • 条件后面要有冒号。
  • 缩进是分支结构的一部分。
  • 先写最简单的两分支，不要一开始堆很多判断。
  • 判断的本质是分流。
今天最该抓住的点： 先把“最小正确写法”做出来，再谈更多变化。只要你能看懂、能改一版、能自己写一个同类小例子，今天就过关了一大半。
