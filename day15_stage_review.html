3. 综合复盘案例

下面这段代码虽然不长，但已经把这一阶段的几个关键点连起来了。

def show_pass_scores(scores): for score in scores: if score >= 60: print("及格分：", score) show_pass_scores([88, 59, 91, 45, 73])
你现在应该开始有这种感觉： 学到这里，你写的已经不只是零散语句，而是一段能接收数据、遍历数据、做判断并输出结果的小程序。

4. 这段代码分别用到了什么

Fn

函数和参数

`show_pass_scores(scores)` 表示把一组分数交给这个函数处理。

List

列表

传进去的是一组分数，而不是单个值。

Loop

循环

`for score in scores` 逐个检查每一个分数。

If

条件判断

只有满足 `score >= 60` 的分数才会被输出。
