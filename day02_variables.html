先别怕英文： 你今天真正要理解的，不是单词本身，而是“给一份数据起名字，以后就能反复拿来用”。

2. 变量是什么

变量可以先理解成一个有名字的盒子。这个盒子里可以放文字、数字，或者以后更多种类的数据。

姓名 年龄 价格 分数
例

生活类比：贴标签的收纳盒

如果你有几个收纳盒，分别写着“文具”“证件”“充电线”，下次找东西时就很快。变量也是一样，程序以后要用这些内容时，直接通过名字去找。

3. 为什么必须学变量

因为程序几乎不可能只处理“写死不变”的内容。很多信息都需要先记住，再继续使用。

No

没有变量会怎样

每次都得重新写一遍数据，代码会很乱，后面要修改也很麻烦。

Yes

有变量会怎样

你只需要改一次变量的值，后面所有使用到它的地方都能跟着变，程序更容易管理。

你可以把变量理解成： 先把信息安顿好，程序后面才能继续做计算、做判断、做展示。

4. 奶茶订单案例

先看一个最贴近生活的例子。你点了一杯奶茶，程序要记住名字、价格和数量。

drink = "奶茶" price = 18 count = 2 print(drink) print(price) print(count)
  • `drink = "奶茶"`：创建一个叫 `drink` 的变量，里面放“奶茶”。
  • `price = 18`：创建一个叫 `price` 的变量，里面放 18。
  • `count = 2`：创建一个叫 `count` 的变量，里面放 2。
  • `print(...)`：把这些变量里的内容输出出来。
这段代码最关键的理解： 左边是名字，右边是内容。程序记住的是“这个名字对应什么值”。

5. 变量怎么写

最基础的写法就是：

变量名 = 值

例如：

name = "小林" age = 18 city = "上海"
注意： 这里的 `=` 不是数学里的“左右相等”，而更像“把右边交给左边”。

6. 常见错误

X1

变量没定义就直接使用

print(name)

如果前面没有 `name = "小林"`，程序就不知道 `name` 是谁。

X2

字符串忘了加引号

name = 小林

文字内容需要写成：

name = "小林"

8. 今日总结

  • 变量 = 有名字的盒子。
  • 写法：`变量名 = 值`。
  • 变量可以保存文字、数字等信息。
  • 变量的值可以被修改。
  • 程序后面可以反复使用这些变量。
今天过关标准： 你能自己写出 3 到 5 个变量，并且分得清“哪个是变量名，哪个是值”，就算真的掌握了今天的内容。
