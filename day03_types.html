1. 今天目标

昨天你学会了“把信息放进变量”。今天的重点是继续追问：这些信息各自属于哪一类？因为只有分清类型，程序才能知道该怎么处理它们。

一句话理解： 数据类型就是“这个值属于哪一类”。比如名字属于文字，年龄属于整数，余额属于小数。

2. 4 个核心类型

S

`str` 字符串

主要表示文字内容。通常会被引号包起来，比如姓名、城市、课程名。

name = "小林"
I

`int` 整数

没有小数点的数字，比如年龄、数量、排名。

age = 18
F

`float` 小数

带小数点的数字，比如价格、余额、身高。

money = 99.5
B

`bool` 布尔值

表示真假判断。只有两个值：`True` 和 `False`。

is_vip = True

3. 用户信息案例

看一个最典型的综合例子：同样都是“用户信息”，其实里面的数据类型并不一样。

name = "小林" age = 18 money = 99.5 is_vip = True
  • `name` 是字符串，因为它是文字。
  • `age` 是整数，因为年龄一般用整数字表示。
  • `money` 是小数，因为余额常常会有小数点。
  • `is_vip` 是布尔值，因为它只回答“是或不是”。

4. 用 `type()` 查看类型

当你不确定一个值到底属于哪一类，可以用 `type()` 来检查。

print(type(name)) print(type(age)) print(type(money)) print(type(is_vip))
记住： `type()` 不是让你背答案，而是一个帮你确认“程序眼里它是什么”的工具。

5. 容易混淆的地方

!

`18` 和 `"18"` 不是一回事

`18` 是整数，`"18"` 是字符串。虽然看起来都像 18，但程序处理它们的方式不同。

为什么这很重要： 后面做输入、运算时，很多报错本质上都和“类型没搞清楚”有关。

6. 今日练习

P1

各写一个例子

分别创建一个 `str`、一个 `int`、一个 `float`、一个 `bool`，再用 `type()` 看它们的类型。

P2

用户信息练习

自己定义姓名、年龄、余额、是否会员这 4 个变量，并判断它们分别属于什么类型。

P3

对比练习

a = 10 b = "10"

观察 `type(a)` 和 `type(b)`，明确它们不一样。

7. 今日总结

  • `str`：文字内容。
  • `int`：整数。
  • `float`：小数。
  • `bool`：真假判断。
  • `type()`：查看数据类型。
今天过关标准： 你能自己写出这 4 种类型各一个例子，并能解释为什么它们不属于同一类。

类型分清楚后，程序就能开始和人互动了。
